# Frecuencia de palabras totales.

Función que cuenta la frecuencia de palabras totales en cada una de las
notas/textos.

## Usage

``` r
acep_frec(x)
```

## Arguments

- x:

  vector de textos al que se le aplica la función de conteo de la
  frecuencia de palabras.

## Value

Si todas las entradas son correctas, la salida sera un vector con una
frecuencia de palabras.

## Examples

``` r
acep_frec("El SUTEBA fue al paro. Reclaman mejoras salariales.")
#> [1] 8
```
